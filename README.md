# ForceX

Aplicação web desenvolvida em **React** para uma academia fictícia chamada **ForceX**, com foco em apresentar planos, informações institucionais, contato e uma calculadora de IMC. O projeto possui uma proposta visual moderna e destaca o uso de tecnologia e inteligência artificial no acompanhamento fitness. :contentReference[oaicite:0]{index=0} :contentReference[oaicite:1]{index=1}

## Funcionalidades

- **Página Inicial (Home)** com banner principal, estatísticas, benefícios da academia e destaque para os planos mais populares. :contentReference[oaicite:2]{index=2}
- **Calculadora de IMC** com campos para peso, altura e impedância opcional, além de classificação automática do resultado. :contentReference[oaicite:3]{index=3}
- **Página de Planos** com alternância entre valores mensais e anuais, exibição de benefícios e diferenciais de cada plano. :contentReference[oaicite:4]{index=4}
- **Página Sobre** com apresentação da academia, missão, visão, valores, linha do tempo e equipe. :contentReference[oaicite:5]{index=5}
- **Página de Contato** com formulário para envio de mensagem. :contentReference[oaicite:6]{index=6}
- **Página de erro 404** para rotas não encontradas. :contentReference[oaicite:7]{index=7}
