# ForceX

Aplicação web desenvolvida em **React** para uma academia fictícia chamada **ForceX**, com foco em apresentar planos, informações institucionais, contato e uma calculadora de IMC. O projeto possui uma proposta visual moderna e destaca o uso de tecnologia e inteligência artificial no acompanhamento fitness. 

## Funcionalidades

- **Página Inicial (Home)** com banner principal, estatísticas, benefícios da academia e destaque para os planos mais populares. 
- **Calculadora de IMC** com campos para peso, altura e impedância opcional, além de classificação automática do resultado. 
- **Página de Planos** com alternância entre valores mensais e anuais, exibição de benefícios e diferenciais de cada plano. 
- **Página Sobre** com apresentação da academia, missão, visão, valores, linha do tempo e equipe. 
- **Página de Contato** com formulário para envio de mensagem. 
- **Página de erro 404** para rotas não encontradas. 

## Tecnologias utilizadas

- React
- JavaScript
- React Router DOM
- CSS

As páginas usam componentes React com importação de arquivos CSS separados por seção/página.
